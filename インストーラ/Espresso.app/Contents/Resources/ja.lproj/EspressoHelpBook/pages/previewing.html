Previewing Web Pages

To add a Web Preview, you need to edit a document using the HTML language by choosing View > Language > HTML. Then, click "Preview" in the toolbar. Alternatively, you can create a blank new Preview in a project by clicking the "New" button in the toolbar and choosing "Live Web Preview".

The Live Preview will automatically refresh when you make changes to local CSS files and HTML pages, for online pages you need to set up an Override using the "Style Sheets" button in the Preview toolbar.
