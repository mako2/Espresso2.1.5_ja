Espresso Help

Editing Code

Espresso features a powerful text editor with many useful tools to quickly change your code and navigate through it. In the toolbar, you will find:

  • An "Action" button that contains context sensitive commands
  • A "Tools" button that shows or hides the Tools sidebar
  • Depending on the current language, context sensitive buttons like "New Tag" and "Un/Comment"

The Tools sidebar holds the following tools:

  • Navigator: a clearly structured outline of your code
  • Quick Publish: if your file is part of a project, you can enable this to publish when saving
  • Snippets: context-sensitive and custom text clips for easy access to often used text
